# Dependency
- stack
- alex
- happy
- bnfc

# Setup
## install stack
```bash
sudo apt install -y stack
```

## build omnirev
1. clone this project
```bash
git clone https://github.com/MountainSeal/omnirev
```
2. change directory
```bash
cd omnirev
```

3. build this project
```bash
stack build
```
